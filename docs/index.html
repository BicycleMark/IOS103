IOS103 - Make your images look great on all devices

Before you begin

This is a Xamarin.iOS class. To complete the coding exercises, you will need a Xamarin.iOS development environment with either a simulator or a device to run the exercise on. Please see the Windows setup or macOS setup pages if you need help installing the Xamarin development environment.

Overview

One key aspect to building a successful application is making it look visually appealing. An easy way to do that is to add images to your application, however, there are a few challenges associated with that. In this course we will learn how to add images to our applications and ensure they look sharp and consistent across all devices.

Objectives

  1. Include an image in your Application Bundle using Build Action
  2. Display an image with UIImageView
  3. Show an image at constant size across devices using points
  4. Provide crisp images across all devices
  5. Organize images using an asset catalog

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Include an image in your Application Bundle using Build Action
    Exercise
    In this exercise, we will add an image into our application and ensure it's Build Action is set to BundleResource.
  2. Display an image with UIImageView
    Exercise
    In this exercise, we will visualize a UIImage in our application.
  3. Show an image at constant size across devices using points
    Exercise
    In this exercise, we will use points to ensure an image is the same size across devices.
  4. Provide crisp images across all devices
    Exercise
    In this exercise, we will add multiple images at different resolutions to allow iOS to select the best for the device.
  5. Organize images using an asset catalog
    Exercise
    In this exercise, we will create an Image Set to group together multi-resolution images.
