IOS103 - Make your images look great on all devices

Goals

The goal of this lab is to create an Image Set to organize multiple images. Image Sets have the benefit of being less error prone because they don't use a file naming convention for organization.

Required assets

This is a continuation of the previous exercise. You can use your existing solution or open the solution in the Exercise 4/Completed folder. The Exercise 5 folder contains a subfolder named Completed with a solution you can use to check your work. Please make sure you have these folders before you begin.

Challenge

  1. Create an Asset Catalog.
  2. Create an Image Set.
  3. Apply an Image Set to a UIImageView.

Steps

Select the Visual Studio environment you want to work with:   macOS or Windows

Create an Asset Catalog

  1. Right-click on the SavingsPlanner project and choose Add > New File....
  2. Select the iOS > Asset Catalog file template.
  3. Name the new Asset Catalog "Assets" and click the New button.

Create an Image Set

  1. In the Asset Catalog, right-click in the left panel and select New Image Set.
  2. Select the Image Set to open it.
  3. Click on the Image Set again and rename it to "DetailedSavings".
  4. Inside the Universal section, click the 1x, 2x, and 3x boxes to navigate to the Exercise 5/Assets folder. Add the corresponding images for each resoultion box. For example, "DetailedSavings@1x.png" will be placed in the 1x box.

Create an Asset Catalog

  1. Right click on the Asset Catalogs folder and select "Add Asset Catalog".
  2. Leave the Asset Catalog as it's default name and select "Add".

Create an Image Set

  1. In the Asset Catalog, select the green add button at the top-left side of the screen.
  2. In the add button dialog, select "Add Image Set". The Image Set should appear in the list below the green add button.
  3. Right click on the Image Set and rename it to "DetailedSavings".
  4. Select the Image Set to open it.
  5. For the 1x, 2x, and 3x boxes, select the browse button and navigate to the Exercise 5/Assets folder. Add the corresponding images for each resoultion box. For example, "DetailedSavings@1x.png" will be placed in the 1x box.

Apply an Image Set

  1. Open the Main.storyboard designer file.
  2. Drag a new UIImageView from the toolbox onto the AboutViewController.
  3. In the Properties Window, select the "Layout" tab.
  4. Set the UIImageView's Width and Height properties to 200.
  5. Now that the size of the UIImageView has changed, you may have to move it into the center of the screen by dragging it. Be sure to keep the UIImageView below the "Savings Planner" label.
  6. Give the UIImageView a name by setting its Name property to "pigImageView".
  7. Enter constraint mode by selecting the "Constraint Editing Mode" button.
    Quick Tip: The "Constraint Editing Mode" button is located at the top-right side of the designer.
  8. With the UIImageView selected add the following constraints:
    • Top spacing constraint: top of UIImageView to bottom of "Savings Planner" label.
    • Horizontal alignment constraint: center of UIImageView to horizontal center of AboutViewController.
      Quick Tip: To add this constraint, grab the center square and drop it on the line that is vertically going through the UIImageView.
    • Constant width constraint
    • Constant height constraint
  9. Open AboutViewController.cs.
  10. Override the ViewDidLoad method and assign the pigImageView's Image property to the DetailedSavings Image Set. Use the UIImage.FromBundle() method and pass in "DetailedSavings" as the parameter.

    Show Code 

    public override void ViewDidLoad()
{
   base.ViewDidLoad();
   pigImageView.Image = UIImage.FromBundle("DetailedSavings");
}

Summary

In this exercise, you added multiple versions of an image at different resolutions.
