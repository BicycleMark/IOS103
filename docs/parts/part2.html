IOS103 - Make your images look great on all devices

Goals

The goal of this lab is to display an image in the Savings application. We're going to add a UIImageView in the Storyboard designer and set the UIImageView's Image property to the image located inside the Resources folder.

Required assets

This is a continuation of the previous exercise. You can use your existing solution or open the solution in the Exercise 1/Completed folder. The Exercise 2 folder contains a subfolder named Completed with a solution you can use to check your work. Please make sure you have these folders before you begin.

Challenge

  1. Create a UIImageView in the Storyboard designer.
  2. Set the UIImageView's Image property.

Steps

Create a UIImageView

  1. Open the Main.storyboard designer file.
  2. In the Toolbox, drag a UIImageView onto the SavingsViewController. Leave the UIImaveView it's default size and shape and place it above the "Starting balance: $" entry.

Set the Image property

  1. Select the UIImageView and locate the Properties Window. In the Properties Window, set the Image property to "Savings.png".
  2. Run the application and verify that the image is being displayed.
    Quick Tip: The image may not be fully visible depending on what iOS simulator you use. We will look at how we can fix this later in the course.

Summary

In this exercise, you created a UIImageView and set it's Image property to display the image located inside the Resources folder.
