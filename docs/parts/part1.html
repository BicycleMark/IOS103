IOS103 - Make your images look great on all devices

Goals

The goal of this lab is to add an image file into your Application Bundle.

You will be working on a supplied Savings calculator application. The Savings application contains all of the required code to estimate investment growth over time. The UI consists of two UIViewControllers, SavingsViewController and AboutViewController. You will add an image file into your application's Resources folder and verify that the Build Action is set to BundleResource. This ensures that the image file is included inside the Application Bundle.

Required assets

The provided Exercise 1 folder contains a subfolder named Start with the starter project you will use for this lab. It also contains a Completed folder with a solution you can use to check your work. Please make sure you have this folder before you begin.

Challenge

  1. Open the provided Savings solution in the Exercise 1/Start folder.
  2. Add an image file into the Resources folder.
  3. Verify the image's Build Action is set to BundleResource.

Steps

Run the starter solution

  1. Open the provided Savings solution in the Exercise 1/Start folder.
  2. Run the app.
  3. The SavingsViewController contains four fields for calculating invenstment growth over time.
  4. Verify that the application is working by entering the sample data and clicking the "Calculate" button.
    • Starting balance: 10,000
    • Interst rate: 6.5
    • Monthly deposit: 100
    • Number of years: 10
    The final balance should be $35,962.15

Add an image file into the Resources folder

  1. Right click on the Resources folder and add an existing item by selecting "Add" followed by "Existing Item...".
  2. Navigate to the Exercise 1/Assets folder and select Savings.png and then select "Add".
  3. Verify that Savings.png is now in the Resources folder.

Verify the image's Build Action

  1. Select the Savings.png file that is located inside the Resources folder.
  2. Locate the Visual Studio Properties Window.
    Quick Tip: If you can't find the Properties Window you can open it by selecting the Visual Studio "View" menu option followed by "Properties Window".
  3. In the Properties Window, find the "Build Action" entry and verify that the value is set to BundleResource.

Summary

In this exercise, you added an image file to the Resources folder of the Savings application. Since we want the image to be packaged into the Application Bundle, we verified that the Build Action was set to BundleResource.
