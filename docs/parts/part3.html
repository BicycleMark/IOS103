IOS103 - Make your images look great on all devices

Goals

The goal of this lab is to define the size of our UIImageView using points. By using points, the UIImageView can use a different amount of pixels depending on the device being used. This allows iOS to make the UIImageView look about the same physical size across devices.

Required assets

This is a continuation of the previous exercise. You can use your existing solution or open the solution in the Exercise 2/Completed folder. The Exercise 3 folder contains a subfolder named Completed with a solution you can use to check your work. Please make sure you have these folders before you begin.

Challenge

  1. Set the UIImageView's size using points.
  2. Add constraints to the UIImageView to keep it's size and position constant.

Steps

Set size using points

  1. Open the Main.storyboard designer file.
  2. Select the UIImageView that's inside the SavingsViewController.
  3. In the Properties Window, select the "Layout" tab.
  4. Set the UIImageView's Width and Height properties to 140.
  5. Now that the size of the UIImageView has changed, you may have to move it back into the center of the screen by dragging it. However, be sure to keep the UIImageView above the "Starting balance: $" entry.

Add constraints to the UIImageView

  1. Select the UIImageView that's inside the SavingsViewController.
  2. Enter constraint mode by selecting the "Constraint Editing Mode" button.
    Quick Tip: The "Constraint Editing Mode" button is located at the top-right side of the designer.
  3. With the UIImageView selected add the following constraints:
    • Top spacing constraint: top of UIImageView to top of parent UIView.
      Quick Tip: The UIImageView is inside another UIView. This UIView is below the "Savings Planner" label, be sure to not pass it while adding this constraint.
    • Horizontal alignment constraint: center of UIImageView to horizontal center of SavingsViewController.
      Quick Tip: To add this constraint, grab the center square and drop it on the line that is vertically going through the UIImageView.
    • Constant width constraint
    • Constant height constraint
  4. Run the application on multiple devices and notice how the UIImageView is in approximately the same positon and size.

Summary

In this exercise, you sized and positioned a UIImageView using points to ensure that it's the same physical size on multiple devices.
