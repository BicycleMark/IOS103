IOS103 - Make your images look great on all devices

Goals

The goal of this lab is to provide an image at multiple resolultions to allow iOS to choose the best for a specific device. The purpose of this is to ensure that the image looks crisp on all devices.

Required assets

This is a continuation of the previous exercise. You can use your existing solution or open the solution in the Exercise 3/Completed folder. The Exercise 4 folder contains a subfolder named Completed with a solution you can use to check your work. Please make sure you have these folders before you begin.

Challenge

  1. Add multiple images at different resolutions.
  2. Run the application on multiple devices.

Steps

Add multiple images at different resolutions

  1. Right click on the Resources folder and add an existing item by selecting "Add" followed by "Existing Item...".
  2. Navigate to the Exercise 4/Assets folder and select Savings@2x.png and Savings@3x.png and then select "Add".
  3. Verify that Savings@2x.png and Savings@3x.png are now in the Resources folder.

Run the application on multiple devices

  1. Run the application on multiple devices and verify that that the image continues to look crisp.

Summary

In this exercise, you added multiple versions of an image at different resolutions.
